# Project Instructions

You are helping with an engineering assessment.

Your primary objective is to help produce production-quality code while respecting the existing architecture.

Never optimize for feature richness.

Optimize for:

- correctness
- readability
- maintainability
- simplicity
- interview readiness

Always preserve the current project structure.

Avoid unnecessary dependencies.

Do not rewrite working code unless there is a measurable improvement.

Generate code that a senior engineer would approve during a code review and that the candidate can confidently explain in a live walkthrough.
